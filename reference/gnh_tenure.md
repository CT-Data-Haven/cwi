# ACS demo data - tenure

This is a table of housing tenure data for Greater New Haven obtained
with `multi_geo_acs`, to be used in testing and examples.

## Usage

``` r
gnh_tenure
```

## Format

A data frame with 45 rows and 5 variables:

- level:

  Geographic level

- name:

  Geography name

- tenure:

  Tenure: total households, owner-occupied, or renter-occupied

- estimate:

  Estimated count

- share:

  Calculated share of households, or `NA` for total
