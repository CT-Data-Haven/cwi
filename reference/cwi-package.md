# cwi: Prep for DataHaven's Community Wellbeing Index

This is a collection of functions for collecting and analyzing Census
data and other sources to write the 2019 Community Wellbeing Index
reports. It also contains reference tables for local
geographies—neighborhoods, regions, etc—and commonly used ACS table
numbers.
