# Common ACS table numbers

Lists of ACS table numbers commonly used by DataHaven.
`basic_table_nums` is used for making short profiles of towns,
neighborhoods, and regions, while `ext_table_nums` is used for
"extended" profiles, such as neighborhood profiles distributed as PDF on
the DataHaven website.

## Usage

``` r
basic_table_nums

ext_table_nums
```

## Format

A named list, where names correspond to abbreviated subjects and string
values correspond to table numbers.
